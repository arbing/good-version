# 从版本对象导出版本压缩包

版本压缩包导出某一个好版本保存时的完整项目文件快照，只包含可直接使用的项目文件，不包含版本历史。导出实现从目标好版本对应的版本对象读取文件并写入 zip，不 checkout、不 reset、不修改当前项目目录；这样即使当前项目目录已被改坏，外置保存数据仍可导出对应好版本。

## Considered Options

- 临时回退当前项目目录再压缩：实现直观，但压缩中断或失败可能改变用户当前文件。
- 创建临时工作区再压缩：不触碰用户目录，但增加临时目录管理和清理复杂度。
- 直接从版本对象写 zip：避免修改用户目录，也支持外置保存数据在项目目录丢失时继续导出。

## Consequences

版本压缩包固定为 `.zip`，压缩包内包一层项目目录。导出保留该好版本中保存过的隐藏文件，只排除 `.git` 版本控制数据。用户选择导出目录后，如果同名 zip 已存在，应用自动生成不重名文件；导出成功后打开导出目录。
